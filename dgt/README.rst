
=========================
APLICACION WEB CON WEB SERVICE
=========================

PROYECTOS
----------------
    * modelo: libreria java (jar) para gestionar el modelo de datos
    * multas_agentes: proyecto java web (war) para gestion de multas de un agente.
    * backend: proyecto java web (war) para gestionar toda la información como administradores (CRUD: agentes, coches etc...)
    * WS_SOAP: Servicio web con SOAP, para poder consultar información de un vehículo.
